## :toolbox: Getting Started

### :bangbang: Prerequisites

### :running: Run Locally

Clone the project

```bash
https://github.com/TakhirKudusov/predict-helper-react-native
```

Install dependencies
```bash
npx expo install
```

Start the development server
```bash
npm start
```
